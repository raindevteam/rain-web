USER GUIDE

This guide aims to be a somewhat detailed overview about the commands and features of Rainbot. However development of the bot is rapid, and and keeping documentation in check can be rather slow, so don't always expect this to be 100% up-to-date with releases.

Commands and Triggers

Commands

Messages proceeded by the bot's nick or the command symbol will be taken as direct commands. For example, if the bot's nick is 'RainBot' and the command symbol in the config is '$', then 'RainBot, ;say this' and '$ ;say this' would both be valid triggers assuming your command prefix has been set to ';'.

If you type just a command name like ';say', then that will also trigger as a command, so you don't necessarily have to use a command line prefix as in the previous examples.

Triggers

Triggers fire separately from commands, which is important to note if you're developing trigger meant to be fired over commands (you would need to use the onCommand property). Triggers will only fire if their appropriate trigger function returns true.

Included Base Modules

The included base modules should help you get started with using the bot in your own channel. The RainCore module also provides some commands to manage the bot. However, if you wish to create your own management module, you are more than welcomed, and if you wish, you can use RainCore as a guide.

RainCore Module

As mentioned, the RainCore module adds some management commands and some somewhat useful commands. It also includes the alias command, so if you remove this module, expect to lose aliasing as well unless reimplemented (and like I said, if you want your own core module, go ahead and use what you want from here in your own).

Command Description
;help [command] Links you to this page, or displays help text of command.
;ver Returns the version of the bot.
;alias name [p1 [,p2]] -> [c1 [, c2]] Aliases command(s) to the given name
;chan leave | join chan[:pass] Allows the bot to join and leave channels.

RainChat Module

The RainChat module provides a few commands for socialness.

Command Description
;say [text] Repeats what you place in for text.
;do [action] Repeats what you place for action, as an action.
;blog Links the blog site for RainBot's development.
;rules Gives you rules, one way or another (For SRQ's Constellation).

RainCal Module

With the power of the mathjs library (http://mathjs.org), RainBot can parse equations and store user defined functions.

Command Description
;cal [equation] Parses the given equation.

RainBrella Module

The RainBrella module implements random utilities.

Command Description
;echo [text] Repeats what you pass in for text (not immediately like say).
;roll [<n1>r [, <n2>d [, <n3>p]]] Rolls for n1 times, with a n2 die in n3 pairs
;jip Evaluates javascript and returns value.

Other Modules

All of the following modules you can find on my Github and use at your leisure.

RainShow Module

The RainShow module provides a command that randomly picks an episode from MLP and can also tell you the title if given a two pair key.

Command Description
;ep [s e] Gives you a random episode to watch.
